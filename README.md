# Pattern Generator in Python

## Description
This project is a Python-based Pattern Generator that creates different star (`*`) patterns using loops, functions, and file handling. The program demonstrates fundamental Python programming concepts by generating various triangle and pyramid patterns and saving the output to a file.

## Features
- Right-Angled Triangle (RAT)
- Left-Angled Triangle (LAT)
- Pyramid Pattern
- Inverted Right-Angled Triangle (IRAT)
- Inverted Left-Angled Triangle (ILAT)
- Inverted Pyramid Pattern
- Uses Functions for modular programming
- Uses Loops for pattern generation
- Uses File Handling to save generated patterns

## Technologies Used
- Python
- Functions
- Loops (`for` loops)
- File Handling

## Output
The generated patterns are stored in an output file using Python file operations.
